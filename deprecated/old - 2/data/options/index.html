  1. This option defines the width of the injected translator. Acceptable values are in the range of 200 to 600. Height of the panel varies based on the content.
  2. This option scales the injected translator to the factor between 0.5 to 1.0. If font-size of the injected translator is too big for you scale it and check the result.
  3. Offset the bubble's position in x direction. This value can be either positive or negative based on direction of the offset.
  4. Offset the bubble's position in y direction. This value can be either positive or negative based on direction of the offset.
  5. If this option is selected, the extension displays a bubble above all your selections to open the translator panel. If the option is unchecked, a new item is added to the context-menu.
  6. If this option is checked, the extension displays the FAQs page after each upgrade to notify you about the recent changes.
  7. Maximum height of the panel. Use zero to disable this option
  8. Styles of this section are added to every page. Make sure to prefix your styles with ".itanywhere-panel" to only target the div and iframe element of this extension
  9. Styles of this section only apply to translate.google.com and translate.google.cn inside the extension's frame element
  10. When translation page using browser-action click, use this engine for translation.
  11. If there is a "Definitions of %s" section, hide the translation section. This is useful to hide the translation section when Google Translator is used for exploring words definition.
  12. If this option is selected, the extension directly opens the translation window when alt key is hold while double-clicking on a word to select it or when alt key is hold while selecting a sentence.
